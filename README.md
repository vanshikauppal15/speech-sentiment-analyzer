# Speech Sentiment Analyzer

This project converts speech to text using OpenAI Whisper and performs sentiment analysis using HuggingFace Transformers.

## Features

- Audio upload
- Speech to text transcription
- Sentiment analysis (Positive / Negative)

## Tech Stack

Python
Flask
OpenAI Whisper
HuggingFace Transformers

## Run Project

pip install -r requirements.txt
python app.py
